"synthetic data generation.R" is applied to generate synthetic data of different configurations.

"Hyper-parameter estimators functions of iETKRR.R" is uesd to provide four Hyper-parameter estimators for iETKRR method, which is from the author of iETKRR method.

"estimation of different methods.R" gives different methods of linear regression for interval-valued data to obtain the corresponding parameters estimated.

"prediction of different methods.R" is applied to calculate four indexes in the test set using the parameters estimated in the training set for each method.

"method compare.R" summarized all ten methods involved in the simulation and present the result of comparison of different methods, with respect to the empirical average and standard deviation of each index.

"an example.R" is an example to explain how to run the code and obtain results.
